# Setup styling:

1. If you have installed prettier plugin in your vscode Disable/Uninstall it.
2. run the following command in your project's root directory
> `npm i -D eslint prettier eslint-plugin-prettier eslint-config-prettier eslint-plugin-node eslint-config-node eslint-config-airbnb-base`
3. Copy `.eslintrc.json` at the root of your project.
